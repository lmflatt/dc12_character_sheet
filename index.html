DC12 Character Sheet Generator

A homebrew module for the beginner-friendly TTRPG system, DC12. This offers an accessible first step into classic TTRPG elements, such as class identity, skill modifiers, and character sheets. Take only a minute to build your character with easy-to-grasp skills and classes. Select "Custom Class" to make your own, or play with no class selected for a non-combat game! Beginner Dungeon Masters, get the most fun out of your games with this Helpful Guide to running DC 12 with the additions from these character sheets!

Disclaimer: This is a fan-made project by a passionate DM Dad unaffiliated with the creators of DC20 and DC12.
